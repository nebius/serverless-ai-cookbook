# Life Science Examples

Domain-specific Serverless AI examples for biology, genomics, chemistry, and molecular simulation. These recipes are nonclinical and research-only unless a README states stricter limits.

## Catalog

| Example | Runtime | What it demonstrates |
|---|---|---|
| [`bionemo-nim-endpoints`](./bionemo-nim-endpoints/README.md) | Serverless AI Endpoints | Deploy BioNeMo/HCLS NVIDIA NIMs with in-region image mirroring, sample structure-prediction requests, readiness and metrics checks, cleanup, and candidate validation notes. |
| [`openmm-simulation`](./openmm-simulation/README.md) | Serverless AI Jobs | Run GPU-backed OpenMM molecular dynamics simulations and persist artifacts to object storage. |
| [`parabricks-deepvariant`](./parabricks-deepvariant/README.md) | Serverless AI Jobs | Run NVIDIA Parabricks DeepVariant genomics workflows with NGC authentication and optional S3-backed inputs and outputs. |
